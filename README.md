1. Bus Ticket Booking (inspired from youtube video):

Bus ticket bookign application (similar to redbus) using Angular 19 with api from "https://www.freeprojectapi.com/".

First page:
displays "From" and "To" dropdown with data from api and "date fo travel".On filling the details it goes to second page.

Second Page:
displays list of buses avaiable for that particular date in that route.These bus details inlcude "Bus Name, fare, timings" are shown based on api call for bus schedules.
On selecting one bus it goes to third page.

Third Page:
displays number of seats aviable and already booked and by entering contact detials for that particular seats tickets are booked.
